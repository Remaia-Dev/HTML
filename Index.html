Sobre Tags Básicas

HTML - Primeiramente temos a tag HTML, que sinaliza que tudo que estiver dentro dela, é um elemento HTML.

HEAD - É a “cabeça” da página. Essa tag faz um pré carregamento de tudo antes de exibir para o usuário. O que está dentro dela não é exibido.

TITLE - O que fica dentro dessa tag, aparece na aba da página como título.

BODY - É o corpo da página. Tudo que fica dentro dessa tag, é exibido para o usuário. Dentro dela colocamos muitas outras tags.

STRONG - Serve para colocar um texto em negrito.

I - Para colocar texto em itálico.

INPUT - Para inserir um campo de digitação na página, que pode ser numérico, de texto, entre outros.

SUB - Este é um elemento inline destinado a marcar fragmentos de texto que devam ser renderizados em alinhamento vertical mais baixo em relação ao texto principal.

SUP - Este é um elemento inline destinado a marcar fragmentos de texto que devam ser renderizados em alinhamento vertical mais alto em relação ao texto principal.

Atributo das Tags

Nada mais é do que propriedades que uma TAG pode ter. Alguns são mais gerais genéricos, ou seja, todas as TAGS terão o que for comandado na TAG, e outros são mais específicos de cada TAG.


Exemplo de atributos Genéricos:

ID - É um identificador do elemento que estiver “abrigando” esse identificador.

STYLE - Tag que é usada para formatar o HTML, muito usado em CSS. É possível mudar cor, tamanho, espaçamento, entre outros.

CLASS - Serve para padronizar um comando determinado dentro da página.

Exemplo de atributos Específicos:

TYPE - Tag específica da tag INPUT. podemos especificar como tipo numérico (number), texto (TEXT), entre outros.

COLOR - Serve para colocar uma cor em determinado elemento.

IMG - Usado para colocar imagem na página, juntamente usado com o atributo SRC, que serve para indicar o endereço da imagem.

Listas ordenadas e não ordenadas

Essas TAGs são muito utilizadas para ajudar na organização de alguns pontos importantes de conteúdo textual que precisam ser listados e destacados. As listas ordenadas podem ser representadas em forma de lista numérica, onde cada ítem da lista é representado por números,. Enquanto a lista não ordenada é representada por símbolos, como por exemplo, bolinhas. Seguem alguns exemplos abaixo:


Listas ordenadas

  1. ítem 1
  2. Ítem 2
  3. ítem 3
  4. Ítem 4

Listas não ordenadas
  • ítem 1
  • Ítem 2
  • ítem 3
  • Ítem 4

Links

Os links são fundamentais para a navegação na internet, pois eles linkam uma página à outra, nos proporcionando a navegação. Os links podem linkar tanto páginas externas, quanto internas, ou seja, dentro de um mesmo site, por exemplo.

Para linkar uma página à outra, é preciso usar a TAG de link "href" junto à TAG "a" de ancoragem. Isso serve para as páginas externas e internas.

Exemplo de Utilização da TAG de link no HTML



Também existem outros tipos de links no HTML que podem ser utilizados, como por exemplo, link de e-mail, telefone, entre outros. Mas, no momento não iremos nos aprofundar nesses outros links, pois ainda não estudamos. Mas vale a pena saber da existência deles. Outro ponto importante que devemos ressaltar, é que para fazer a ancoragem de um link que irá direcionar para uma página dentro do mesmo site, precisamos criar uma nova página além da página principal (home), por exemplo: se for uma página falando sobre a empresa, podemos criar uma página SOBRE, se for uma página com os contatos da empresa, podemos fazer uma página CONTATOS, e por aí vai. Depois das páginas criadas, iremos usar a TAG de link como vimos acima, mas no lugar do endereço, iremos colocar o endereço das páginas, que devem estar no repositório da página HOME.

Exemplo de link para uma Página Interna
